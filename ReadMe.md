## 1.1 如何运行？
1. 进入frontend文件夹，输入`npm i` 安装依赖包
2. `npm run start` 将react项目run起来即可
3. 如果在调试前端功能的时候，只需要跑`react`项目即可，无需跑后端
4. 如果需要测试前后端项目的时候，需要进入`backend`文件夹，将`app.py`文件跑起即可
